# pattern
Java设计模式   

Java设计模式学习中实现的各个实例  
adapter------适配器模式  
bridge-------桥接模式  
composite----组合模式  
decorator----装饰模式  
factory------工厂类模式  
flyweight----状态模式  
observer-----观察者模式（自定义实现）  
observer2----观察者模式（java.util.Observable类  java.util.Observer接口 实现）  
staticProxy--静态代理模式  
dynamicProxy-动态代理模式  
singleton----单例模式（饿汉式）（懒汉式）（静态内部类模式）（枚举式）  
state--------状态模式  
template-----模板模式
